# Game Config Generator

This project is a web application that allows users to generate configuration files for popular games by selecting a game and customizing settings such as graphics, controls, and mods.

## Features
- Select a game from a list
- Customize settings (graphics, controls, mods, etc.)
- Download the generated config file

## Tech Stack
- Frontend: React
- Backend: Node.js (Express)

## Getting Started

### Prerequisites
- Node.js (v16 or higher) and npm installed

### Setup
1. Install dependencies for both frontend and backend:
   - `cd client && npm install`
   - `cd ../server && npm install`
2. Start the backend server:
   - `cd server && npm start`
3. Start the frontend app:
   - `cd client && npm start`

The frontend will run on http://localhost:3000 and the backend on http://localhost:5000 by default.
